## 一、项目概述

### 1.1 定位
**童心惠民** —— 公益心理咨询 + 参政议政数据支撑平台

通过公益心理咨询服务积累数据，定期向政府/政协/民盟提交参政议政提案报告，用数据为社会治理提供支撑。

### 1.2 使命
让公益心理咨询走进社区，用专业力量守护心理健康，同时为政府提供真实有效的社会心理数据。

### 1.3 我们是谁
童心惠民是由济南市中茉莉心理咨询工作室发起的公益心理服务平台，致力于将专业心理咨询服务带入社区，用温暖和专业守护每一颗需要被倾听的心。

### 1.4 我们做什么
提供免费/低门槛的心理咨询服务，同时系统化积累公益服务数据，定期向政府/政协/民盟提交参政议政提案报告，让真实的心理健康数据成为社会政策制定的依据。

### 1.5 为什么存在
我们相信：心理健康是社会治理的重要组成部分。每一个被看见的数据背后，都是一个真实的人、一段真实的困惑、一个真实的需要。

### 1.6 我们的目标
让公益心理咨询不再遥不可及，让每一份真实的数据都能推动一点点改变。

### 1.7 核心人物
- **王琳（Ceci）** —— 发起人，精神动力学派心理咨询师及督导师
- **玩偶卡卡（Kaka）** —— AI运营中心，负责数据整理/报告生成/日常运营

### 1.8 服务对象
- 社区居民（婚恋困惑、家庭矛盾、职场压力）
- 高校学生（成长困惑、人际关系、生涯规划）
- 特殊群体（儿童/老年人/残障人士/留守儿童）

### 4.3 数据库策略（重要）
| 目录 | 用途 | 是否上云 |
|------|------|---------|
| /db/ident/ | 去标识化数据（供报告使用） | ✅ 可上云 |
| /db/full/ | 全量原始数据（严格保密） | ❌ 本地存储 |

**去标识化规则：**
- 姓名：仅存姓氏
- 电话：仅存尾号4位
- 地址：仅存区级

## 五、数据看板（Dashboard）

**核心指标：**
- 服务人次（按月/季度统计）
- 问题类型分布（婚姻问题/亲子关系/职场压力/其他）
- 高风险个案预警（量表总分超标自动红色标记）
- 来源渠道分析（网站/热线/学校/社区）
- 志愿者服务时长

**提案报告体例：**
```
【封面】项目名称 | 日期 | 报送单位

一、基本情况（项目背景 + 实施情况）
二、调研数据（去标识化 + 可量化）
三、问题分析（当前困境 + 原因剖析）
四、对策建议（1.2.3. 具体政策建议）
五、下一步计划
```

> 版本：v1.0 | 2026年4月

## 一、使命与定位

### 使命
**以家庭为最小单位，守护青少年心理健康，促进家庭系统和谐。**

### 核心价值观
- 🎯 **专业**：以精神动力学为基础，结合系统式家庭治疗
- 💚 **公益**：服务真正需要帮助的家庭，不设门槛
- 🤝 **信任**：建立可信赖的专业形象，数据透明可查
- 🏠 **家庭视角**：以家庭系统看待问题，而非单一成员

### 差异化定位
| 普通NGO | 童心惠民 |
|---------|---------|
| 只服务青少年 | 服务"家庭系统"（老人+成人+孩子）|
| 单次咨询 | 持续随访+家庭关系追踪 |
| 疾病视角 | 家庭健康促进视角 |
| 无数据 | 可量化社会效益报告 |

## 二、市场机会

### 政策顺风（2022-2026）
- 《家庭教育促进法》（2022）—— 要求街道/社区配备家庭教育指导服务
- 《未成年人保护法》修订 —— 强制报告制度，基层需要专业支撑
- 《关于加强新时代老龄工作的意见》—— 老年人心理健康服务纳入基本公共服务
- "社会心理服务体系建设"写入十四五规划

### 市场需求
- 中国6-18岁青少年心理问题检出率持续上升（卫健委数据：约3000万）
- 双职工家庭中，"隔代育儿"带来的亲子/婆媳/老人孤独问题日益突出
- 认知症患者家庭照护者（主要是老人）严重缺乏心理支持
- 政府购买服务预算持续增加

## 三、服务架构

### 分层服务体系

```
┌─────────────────────────────────────┐
│  L4 深度服务（高风险家庭）           │
│  系统式家庭治疗 + 危机干预           │
├─────────────────────────────────────┤
│  L3 专项服务（有明确需求）           │
│  家长课堂/亲子训练营/认知症家属支持  │
├─────────────────────────────────────┤
│  L2 筛查服务（AI+人工初筛）         │
│  家庭关系评估/风险分级/随访提醒      │
├─────────────────────────────────────┤
│  L1 公益托底（零门槛）              │
│  热线/AI机器人/公益科普              │
└─────────────────────────────────────┘
```

### 核心服务模块

#### 1. 青少年心理服务
- 危机干预（评估+转介+随访）
- 学业压力/厌学/亲子冲突
- 多动/注意力/情绪问题
- 网络成瘾

#### 2. 家庭关系服务
- 亲子沟通指导
- 隔代育儿冲突调解
- 婆媳/婚姻关系支持
- 离异/再婚家庭调适

#### 3. 老年人心理服务
- 认知症照护者支持（核心差异化）
- 老人孤独感干预
- 老年抑郁/焦虑
- 丧亲/哀伤辅导

## 四、商业模式

### 四条收入腿

| 收入来源 | 占比目标 | 说明 |
|---------|---------|------|
| 政府购买服务 | 50% | 街道/社区/妇联/民政 |
| 企业EAP | 25% | 员工家庭心理健康福利 |
| 家庭付费 | 15% | 课程/训练营/咨询 |
| 基金会/捐赠 | 10% | 公信力背书 |

### 政府采购切入路径
```
第一步：免费服务（建立信任）
        ↓
第二步：数据积累（证明效果）
        ↓
第三步：形成报告（参政议政）
        ↓
第四步：申请购买（正式合作）
```
